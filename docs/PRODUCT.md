# Expense Tracker — 产品说明文档

_版本：1.0 · 更新：2026-05-21_

---

## 1. 产品概述

**Expense Tracker** 是一款面向香港及华语用户的智能记账 App。用户可通过手动输入或拍照识别收据快速记录支出，查看月度报表，并在订阅 AI Pro 后获得自动分类、消费洞察与云端同步能力。

**产品定位：**

- 面向个人用户的轻量记账工具
- 香港 AI 创业教育课程（OPC/21days）的全栈教学示例
- 可演进至 App Store / Google Play 上架的 MVP 原型

**核心理念：** 记账要「快、准、有洞察」—— 3 秒内完成一笔，OCR 辅助但不替用户做决定，AI 在合适时机给出建议而非强制自动化。

## 2. 目标用户

| 用户群 | 痛点 | 产品价值 |
|--------|------|----------|
| 日常记账用户 | 手动输入繁琐、易漏记 | 拍照 OCR + 快速记一笔入口 |
| 香港本地用户 | 的士小票、电子支付截图格式多样 | 针对 HK$ 等本地格式的 OCR 候选识别 |
| 希望了解消费结构的用户 | 只看数字缺乏行动指引 | 月度饼图 + AI 月报洞察 |
| 多设备用户 | 换机丢数据 | AI Pro 云同步 + 恢复购买（规划中） |

---

## 3. 产品原则

1. **无注册门槛** — 不要求邮箱/手机号，打开即用
2. **离线优先** — 本地 SQLite 为主存储，无网络也能记账
3. **OCR 不猜、不蒙** — OCR 负责「看见」，用户负责「判断」
4. **AI 负责建议** — 分类与洞察需用户确认，高置信度可一键采纳
5. **双端一致** — iOS / Android 同一套后端，不依赖 iCloud

## 5. 用户旅程

### 5.1 首次使用（免费用户）

```
打开 App
  → 后台匿名登录（用户无感）
  → 浏览空支出列表
  → 点「记一笔」
  → 手动输入 或 拍照选收据
  → [OCR] 底部弹窗展示金额候选 → 用户点选确认
  → 选类别、填备注、保存
  → 回到列表 / 切到「报表」看月度分布
```

### 5.2 OCR 记一笔（核心体验）

```
选图 → 本地 OCR 扫描（无需网络）
    → 展示多个金额候选（含 Total 行优先、疑似日期警告）
    → 用户选择 或 手动输入
    → 填入金额字段 → 继续完成记账
```

**设计原则：** 即使 OCR 读错（如 62.60 → 52.60），用户也能从候选列表中纠正，不会被自动填入错误金额。

### 5.3 AI Pro 用户（目标体验）

```
完成金额确认
  → 自动触发 LLM 分类建议（静默，无需额外点击）
  → 底部卡片：「餐饮 ☕ 置信度 92%」
  → [确认] 直接入账  /  [修改分类] 手动调整
  → 数据同步至云端（多设备可见）
  → 月末查看 AI 生成的消费洞察摘要
```

## 8. 关键交互说明

### OCR 金额候选弹窗

| 元素 | 说明 |
|------|------|
| Total 行 | `isLikelyTotal = true`，置顶推荐，含 total/总计/实付等关键词 |
| 普通候选 | 含 HK$ 前缀或上下文行文字 |
| 疑似日期 | 1900–2100 且 .00 结尾，标注「不推荐」 |
| 置信度标签 | ≥ 0.8「很确定」，否则「不太确定」（待上线） |
| 手动输入 | 候选均不对时可兜底输入 |

### AI 分类建议卡片（Pro）

- 展示推荐类别、置信度、简短理由
- 置信度 ≥ 90% 时可一键确认入账
- 低置信度需用户显式确认

## 9. 数据与隐私

| 项目 | 说明 |
|------|------|
| 本地数据 | 存储于设备 SQLite，用户可随时 CSV 导出 |
| 云端数据 | 仅 Pro 用户、且配置 Supabase 时同步 |
| 用户身份 | Supabase 匿名 Auth，无 PII（邮箱/手机） |
| OCR 图片 | 本地处理，不上传服务器 |
| AI 请求 | 仅发送金额、类别、备注等结构化文本至 LLM API |

## 10. 竞品差异与卖点

| 维度 | 传统记账 App | Expense Tracker |
|------|-------------|-----------------|
| 注册 | 常要求邮箱/微信 | 无注册，打开即用 |
| OCR | 自动填金额，易错 | 候选列表 + 用户确认 |
| AI | 部分无或仅海外 | 本地化分类 + 月报洞察 |
| 跨平台云同步 | 常依赖 iCloud | Supabase 双端一致 |
| 离线 | 部分强依赖网络 | SQLite 离线优先 |

**一句话卖点：** 「拍一下、选一下、记完了」—— 比手动快，比全自动准。
